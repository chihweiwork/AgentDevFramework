# Agent-1: Example Agent

This is an example agent that demonstrates the basic structure.

## Purpose

A simple demonstration agent that can echo messages.

## Available Tools

- `hello_world`: Echoes a message back to the user with timestamp

## Usage

```
Use hello_world tool with message "Hello from Agent-1"
```

## Configuration

- Working directory: `~/code/agent-1`
- Tools directory: `~/code/agent-1/tool`
- Scripts directory: `~/code/agent-1/script`
